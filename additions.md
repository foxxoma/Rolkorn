# Столкновения

1. Для слабых противников рекомендую использовать сложность во время сражения; Для сильных - механику противостояния.
2. Если при провале бросков игрок не получает ранение - раунды жизни не тратятся (например при скрытой стрельбе), но привлекает на себя внимание.

**Сопротивление**: уменьшает входящий урон раундам жизней вдвое 
**Уязвимость**: увеличивает входящий урон раундам жизни вдвое

# Эпические угрозы

Это может быть что угодно: босс, обрушение здания, нашествие...
Определим количество фаз (допустим 4). Фаза - это особое событие на сцене.
Каждая фаза имеет свою сложность (20, 15, 17) прокиньте или заранее запишите ее.
Так же вы можете уточнить сколько раундов длится фаза, что бы преодолеть ее или выжить в течении этого времени.

Если игрок выбрасывает больше сложности идет не большое разветвление: 
1. он прерывает фазу - она заканчивается для всех
2. он спасается - не бросает напряжение

При броске напряжения тот же эффект:
1. создал помеху фазе - сложность упала для всех
2. создал преимущество для себя - сложность упала только для тебя

Последнюю фазу сделайте так, что бы ее преодоление ставило точку.

Например: Нападение великана в ветхом здании
Угрозы
1. Разбивает колонны, в здании, что бы достать вас (сложность 17)
2. Мощные удары по земле сбивающие с ног, на 3 удар проваливается пол (сложность 17)
3. Нижний этаж затапливается, нужно выбираться (16)
4. Помешать великану выбраться (15)

# Защитные эффекты

Броня, щиты... дают дополнительные "жизни" (раунды Напряжения), но создает сложности. Дополнительные раунды восстанавливаются только после починки.

Пример:

| Тип брони    | дополнительные раунды при защите | сложность действия при помехе |
| ------------ | -------------------------------- | ----------------------------- |
| **Кожа/щит** | +1                               | +1                            |
| **Кольчуга** | +2                               | +3                            |
| **Латы**     | +3                               | +5                            |

# Продолжительные эффекты

Отравление, обморожение, заражение, истощение... делаем через шкалы 3-5 ячеек.

Влияние: нарративный эффект; уменьшение и увеличение проверок сложности.

Пример: Отравление  
- Тригеры:  
	- Усугубление: отсутствие лечения в течении часа, повышение дозировки
	- Восстановление: противоядия
- Изменения  
	- 1. Ухудшение координации (сложность +1)
	- 2. Рвота  (сложность +3)
	- 3. Обезвоживание   (сложность +5)
	- 4. Паралич
	- 5. Смерть

# Магия
делаем ячейки магии 5, 10, 15 или по аналогии со шкалой: 5 ячеек с делениями.
### Параметры:
- **Бонус при броске** = бонусу биографии
- **Длительность заклинания** = опционально
	*рекомендация: бонусу биографии (часов / использованный / проверок)*
- **Дополнительные раунды при защите** = опционально
	*рекомендация: бонусу биографии*

Сотворение тратит одну ячейку.

Для некоторых заклинаний можно предусмотреть возможность увеличения мощности, за счет траты большего числа ячеек, !но только к одному параметру!
